# SEN306 - Software Construction

## 📘 Course Overview
SEN306 (Software Construction) focuses on the principles, techniques, and tools used in building high-quality software systems. The course emphasizes writing clean, maintainable, and efficient code while applying best practices in modern software development.

---

## 🎯 Objectives
By the end of this course, you should be able to:

- Understand core software construction concepts
- Apply coding standards and best practices
- Design and implement modular and reusable code
- Use version control systems effectively
- Debug, test, and maintain software systems
- Work with development tools and environments

---

## 🛠️ Technologies & Tools
This course may involve the use of:

- Programming Languages: Java, C++, or others
- Version Control: Git & GitHub
- IDEs: Visual Studio Code, IntelliJ IDEA
- Build Tools: Maven, Gradle, or Ant
- Debugging Tools
